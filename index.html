User Guide

  • Left-click the waveform to set the subtitle start time.
  • Right-click the waveform to set the subtitle end time.
  • Drag marker borders on the waveform to adjust timestamps.
  • Press Enter to save a row; Shift + Enter adds a line break.
  • Click a saved row to select it, then use Insert Before, Insert After, or Command + Delete.
  • Import existing .srt/.vtt files, then export finished subtitles from the Export Subtitles menu.

Shortcuts

  • space to play/pause when you are not typing.
  • ⌘ + p to replay the selected segment while editing text.
  • ⌘ + ← / → to move the waveform window backward or forward.
  • ⌘ + + / - to zoom the waveform in or out.
  • ⌘ + z to undo; ⌘ + shift + z to redo.
  • ⌘ + s to save the local draft immediately.
  • The next subtitle draft starts where the previous row ends.
  • On the waveform: pinch/open with two fingers zooms; two-finger left/right moves through time.
